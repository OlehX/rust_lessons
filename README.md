# rust_lessons

Contains my Rust lessons

## Prerequisites

Ensure you have [Rust and Cargo](https://www.rust-lang.org/tools/install) installed on your system.

## Usage

### Build
To compile the project, run:
```bash
cargo build
```

### Run
To execute the project, run:
```bash
cargo run
```

### Test
To run the tests, use:
```bash
cargo test
```
